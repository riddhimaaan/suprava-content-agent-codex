## Key Patterns to Extract

| Element | What the top examples do |
|---|---|
| **Hook** | First line is the entire hook — no warmup sentence |
| **Numbers** | Specific: "$500 workflows", "600 → 6K", "13 days", "1000+ workflows" |
| **Social proof** | Results before explanation |
| **Bullets** | Used for feature lists and step lists, never for main ideas |
| **CTA** | Always specific: what to comment, what you'll get, the requirement |
| **Length** | Short posts hit hard. Longer posts justify length with clear value per line |
| **Tone** | Lowercase openers, contractions, informal — feels real, not polished |
