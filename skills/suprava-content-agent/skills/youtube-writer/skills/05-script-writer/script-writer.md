# YouTube Script Writer

Write full YouTube video scripts section by section — human-sounding, direct, practitioner tone — optimized for spoken delivery, retention, and SEO keyword placement.

## Framework Reference

Before writing any section, read `masterframework.md` in this folder.

That file contains the structural frameworks derived from high-performing YouTube videos — hook patterns, section rhythms, transition rules, recap formats, and CTA structures. Every script section must be written using one of those frameworks. Do not invent a new structure. Pick the framework that fits the section type and apply it.

## Channel Context

**Niche:** B2B marketing, lead generation, AI automation, cold outreach, client acquisition, offer creation
**Audience:** Beginner to intermediate B2B agency owners, cold outreach practitioners, SaaS founders
**Channel Voice:** Direct, no-fluff, practitioner perspective, honest about limitations, systems-focused

## Script Writing Rules

### Voice and Tone

**Always:**
- Write like one person talking directly to another
- Use short sentences — 10–15 words max per sentence
- Use "you" constantly — the viewer is the subject of the script
- Name specific problems, not abstract concepts
- Speak in plain English — no jargon unless immediately explained
- Sound like a practitioner, not a professor

**Never:**
- "In today's video we're going to be exploring..."
- "That's a great question"
- "Make sure to like and subscribe before we dive in"
- Passive voice: "it was found that" → "I found"
- Vague openers: "Many people struggle with..." → name the exact struggle
- Filler transitions: "Moving on to our next point..." → just move to the next point

### Sentence and Paragraph Structure

- Max 3 sentences per paragraph block
- Every paragraph = one idea
- One-sentence paragraphs are fine — use them for emphasis
- Use line breaks to control pacing for the speaker
- Mark pauses where the speaker should breathe or pause: [PAUSE]

### Keyword Placement Rules

Primary keyword: Use in first 60 seconds of spoken content, naturally
Secondary keywords: Weave into explanations naturally — never force them
Never: Stuff keywords unnaturally into sentences

### Section-by-Section Writing Process

Write one section at a time. After each section, stop and wait for user approval before writing the next.

Order:
1. Hook
2. Context Bridge
3. Core Section 1
4. Core Section 2
5. [continue per outline]
6. Recap
7. CTA

### Hook Writing (Priority Section)

The hook must:
- Start mid-thought — as if the conversation is already happening
- Name a specific, painful, recognizable situation in the first sentence
- Create tension that can only be resolved by watching the video
- Run 45–60 seconds when spoken at natural pace (~100–120 words)

Hook structure:
1. The pain statement (1–2 sentences)
2. The surprising or counterintuitive reality (1–2 sentences)
3. The promise (1 sentence — what this video gives them)
4. The implicit "keep watching" (1 sentence)

Example Hook (Cold Email topic):
"You wrote a decent cold email. You personalized the opening. You kept it short. You even tested the subject line.

And you still got zero replies.

Here's the problem — and most people never figure this out — the email isn't the issue. The offer is.

In this video I'm going to show you exactly what's broken and how to fix it. And if you've been sending cold emails with barely any replies, this will tell you why."

### Core Section Writing

Each core section follows this rhythm:

1. **Section opener** — State the point directly. No windup.
2. **Explanation** — Why this is true. Keep it brief.
3. **Example or scenario** — Make it concrete. Use "imagine" or "let's say" to set up scenarios.
4. **Consequence** — What happens if you ignore this point.
5. **Micro-action** — One specific thing the viewer can do with this information.

Section length: 250–400 words per section (2–3 minutes spoken)

### Transitions Between Sections

Do not use generic transitions. Each transition must:
- Reference what was just said
- Create curiosity about what's next
- Take 1–2 sentences maximum

Examples:
- "So that's the first reason. But here's where most people make the bigger mistake."
- "Once you fix that, you run into a different problem — and this one is harder to spot."
- "That covers the targeting side. Now let's talk about what you're actually saying in the email."

### Recap Writing

The recap is not a summary — it's a compression of the most important points.

Format:
- "So here's what we covered:"
- Point 1 in one sentence
- Point 2 in one sentence
- Point 3 in one sentence
- Single most important takeaway: "If you do nothing else from this video, do [X]."

### CTA Writing

One CTA. Spoken naturally, not as a separate formal announcement.

Direct ask format:
"If this helped, subscribe — I post [content type] every [frequency] for [audience type]."

Comment engagement format:
"Drop [specific thing] in the comments — I read all of them and I'll tell you [specific value]."

Related video format:
"If you want to go deeper on [related topic], watch this next — [point to video]."

## Quality Checklist Per Section

Before outputting any section:
- ✓ Opens with direct statement — no windup
- ✓ Sentences are short and spoken-language natural
- ✓ Contains at least one concrete example or scenario
- ✓ No filler phrases or generic transitions
- ✓ Keyword placement is natural — not forced
- ✓ Ends with a clear transition or close
- ✓ Word count matches target duration (130 words ≈ 1 minute spoken)
